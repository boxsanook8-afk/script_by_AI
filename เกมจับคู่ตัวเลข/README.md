# เกมจับคู่ตัวเลข

คำอธิบายสั้น ๆ
โปรเจกต์นี้เป็นเกมจับคู่ตัวเลขที่แสดงตัวเลขใน 3 รูปแบบ: ตัวเลขฮินดู-อารบิก (Arabic numerals), ตัวเลขไทย (Thai digits), และคำอ่านเป็นตัวหนังสือ (คำอ่านตัวเลขเป็นภาษาไทย) — ผู้เล่นต้องคลิกเลือกไพ่จากทั้ง 3 คอลัมน์ให้ตรงกัน

ไฟล์หลัก
- `code_artifact.html` — หน้าเว็บของเกม (เปิดได้โดยตรงในเบราว์เซอร์)
  https://github.com/boxsanook8-afk/script_by_AI/blob/5dc4359b5adc5abf94fa3cfe5fadba3b86646589/%E0%B9%80%E0%B8%81%E0%B8%A1%E0%B8%88%E0%B8%B1%E0%B8%9A%E0%B8%84%E0%B8%B9%E0%B9%88%E0%B8%95%E0%B8%B1%E0%B8%A7%E0%B9%80%E0%B8%A5%E0%B8%82/code_artifact.html

วิธีใช้งาน (เรียกใช้งานแบบง่าย)
1. ดาวน์โหลดหรือเปิด `code_artifact.html` ในเว็บเบราว์เซอร์ (ดับเบิลคลิกหรือเปิดด้วยเมนู Open File ของเบราว์เซอร์)
2. ปรับการตั้งค่าที่ส่วนบน (จำนวนข้อ, ขนาดตัวเลข) แล้วกด “สร้างเกมใหม่”
3. คลิกเลือกไพ่จากทั้ง 3 ฝั่ง (อารบิก ↔ ไทย ↔ ตัวหนังสือ) ให้เป็นเลขชุดเดียวกันเพื่อจับคู่ให้ครบ

ฟีเจอร์เด่น
- สุ่มตัวเลขตามจำนวนและขอบเขตที่กำหนด
- แสดงผลเป็นตัวเลขฮินดู-อารบิก, ตัวเลขไทย, และคำอ่านเป็นตัวหนังสือ
- เอฟเฟกต์เมื่อจับคู่ถูก/ผิด และเอฟเฟกต์ confetti เมื่อทำสำเร็จ
- ปุ่ม Export to Word เพื่อดาวน์โหลดใบงานพร้อมเฉลยเป็นไฟล์ .doc

โครงสร้างไฟล์ (ในโฟลเดอร์นี้)
- `code_artifact.html` — หน้าเกมทั้งหมด (HTML + CSS + JS)
