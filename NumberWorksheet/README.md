# แบบฝึกหัด ตัวเลขไทยและอารบิก (Number Worksheet)

โปรเจกต์นี้เป็นหน้าเว็บ HTML สำหรับสร้างแบบฝึกหัดการอ่านและเขียนตัวเลข (ตัวเลขฮินดู-อารบิก, ตัวเลขไทย และคำอ่านเป็นภาษาไทย) โดยออกแบบให้ครูหรือผู้ใช้สามารถ:
- สร้างข้อแบบฝึกหัดได้หลายข้อ
- สุ่มตัวเลขตามช่วงที่กำหนด
- ปรับให้ซ่อน/แสดงเฉพาะคอลัมน์ที่ต้องการ (จำนวน / อารบิก / ไทย / ตัวหนังสือ)
- ดาวน์โหลดแบบฝึกหัดเป็น PDF หรือไฟล์ Word
- ปรับแต่งต้นฉบับได้ง่าย (HTML + JavaScript เท่านั้น)

ไฟล์หลัก
- number_worksheet_system.html — หน้าเว็บหลักที่มี UI, การสุ่มตัวเลข, การซ่อน/แสดงคอลัมน์ และฟังก์ชันการส่งออกเป็น PDF/Word

คุณสมบัติเด่น
- อินเทอร์เฟซเป็นภาษาไทย เหมาะสำหรับครูผู้สอนระดับประถมศึกษา
- ปุ่มสุ่มตัวเลขที่กำหนดช่วง (min / max)
- ปุ่มสุ่มซ่อนช่อง (สุ่มให้เหลือคำใบ้เพียงคอลัมน์เดียวต่อข้อ)
- สามารถเพิ่ม/ลบแถวข้อแบบฝึกหัดได้
- รองรับการพิมพ์และการบันทึกเป็น PDF (html2pdf) และส่งออกเป็นไฟล์ .doc (สำหรับ Microsoft Word)

การใช้งาน (รวบรัด)
1. ดาวน์โหลดหรือคัดลอกไฟล์ number_worksheet_system.html ไปไว้ในเครื่องของคุณ
2. เปิดไฟล์ด้วยเว็บเบราว์เซอร์ (เช่น Chrome, Firefox) — ไม่จำเป็นต้องมีเซิร์ฟเวอร์
3. ปรับจำนวนข้อโดยกด + เพิ่มข้อใหม่ หรือลบแถวตามต้องการ
4. ป้อนหรือกดปุ่ม "สุ่มตัวเลข" เพื่อเติมค่าอัตโนมัติ (ตั้งช่วง min / max ก่อน)
5. กด "สุ่มซ่อนช่อง" เพื่อสร้างแบบฝึกหัดที่นักเรียนต้องเติมคำตอบ
6. กด "ดาวน์โหลด PDF" หรือ "ดาวน์โหลด Word" เพื่อส่งออกไฟล์สำหรับแจกให้นักเรียน

ปรับแต่งได้อย่างไร
- ค่าเริ่มต้นของแถว (rows) อยู่ในสคริปต์ JavaScript — แก้ไขอาเรย์ `rows` เพื่อเปลี่ยนชุดตัวอย่างเริ่มต้น
- ฟังก์ชันการแปลงตัวเลขเป็นคำอ่านไทยอยู่ที่ `numToThaiText()` — สามารถปรับตรรกะหรือเพิ่มการรองรับจำนวนใหญ่ขึ้นได้
- ใช้ Tailwind CSS และไอคอน Lucide ที่โหลดจาก CDN — ถ้าต้องการใช้งานแบบออฟไลน์ ให้ดาวน์โหลดไฟล์ CSS/ไลบรารีมาเก็บไว้ในโปรเจกต์

การนำไปใช้ในชั้นเรียน
- ครูสามารถสร้างแผ่นงานที่มีระดับความยากต่างกันโดยปรับช่วงการสุ่มตัวเลข
- พิมพ์ออกเป็นเอกสารแจก หรือส่งเป็นไฟล์ PDF ให้ผู้ปกครอง/นักเรียน
- ใช้โหมดซ่อนช่องเพื่อฝึกการคิดและเปรียบเทียบรูปแบบการอ่านตัวเลข

ปัญหาที่อาจเจอและวิธีแก้ไขเบื้องต้น
- ปัญหาการแสดงผลตัวอักษรไทยใน Word: ฟังก์ชัน export ใส่ BOM (\ufeff) เพื่อช่วยให้ Word แสดงภาษาไทยได้ถูกต้อง หากยังมีปัญหา ให้ตรวจสอบฟอนต์บนเครื่อง
- หากไอคอนไม่แสดง ให้เช็คการเชื่อมต่อ CDN ของ Lucide หรือเปลี่ยนเป็นไอคอนชุดอื่น
- หากการบันทึก PDF เกิดปัญหาในบางเบราว์เซอร์ ให้ทดลองอัปเดตเบราว์เซอร์หรือใช้เมนูพิมพ์ของเบราว์เซอร์เป็นทางเลือก
