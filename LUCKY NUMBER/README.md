# LUCKY NUMBER

โปรเจกต์ LUCKY NUMBER เป็นโค้ด HTML สำหรับสร้างหน้าเว็บสุ่มเลขนำโชค (เลขนำโชค, เลขสลาก, หรือเลขเด็ด) ที่ออกแบบโดยปัญญาประดิษฐ์ (AI) และเก็บไว้ในคลังโค้ดของผู้ใช้

## คำอธิบาย
- เวอร์ชันนี้เป็นหน้า HTML แบบง่ายที่แสดงปุ่มเพื่อสุ่มตัวเลขนำโชคตามช่วงที่กำหนด (เช่น 1–99 หรือ 1–1000)
- เหมาะสำหรับทดลองใช้งาน หรือเป็นตัวอย่างการใช้ JavaScript สุ่มตัวเลขและแสดงผลบนหน้าเว็บ

## คุณสมบัติ
- เลือกช่วงตัวเลข (min / max)
- กดปุ่มเพื่อสุ่มเลขใหม่
- แสดงเลขที่สุ่ม พร้อมเอฟเฟกต์หรือการจัดรูปแบบ (ขึ้นอยู่กับโค้ด HTML/JS ที่มี)

## วิธีใช้งาน
1. เปิดไฟล์ HTML (เช่น lucky_number.html หรือไฟล์ที่มีในโฟลเดอร์นี้) ด้วยเว็บเบราว์เซอร์ (ดับเบิลคลิกหรือลากไฟล์ไปยังเบราว์เซอร์)
2. ปรับช่วงตัวเลขหากต้องการ
3. คลิกปุ่ม "สุ่ม" เพื่อดูเลขนำโชค

## โครงสร้างไฟล์ (ตัวอย่าง)
- lucky_number.html — หน้า HTML หลัก (ไฟล์ตัวอย่างที่รวม CSS/JS ไว้ภายใน)
- assets/ — โฟลเดอร์เก็บไฟล์สไตล์หรือภาพ (ถ้ามี)

## การแก้ไขและปรับแต่ง
- สามารถเพิ่มเอฟเฟกต์ภาพด้วย CSS หรือเพิ่มการสุ่มหลายตัวเลขครั้งเดียว
- หากต้องการเก็บประวัติเลขที่สุ่ม ให้เพิ่มการเชื่อมต่อกับ localStorage หรือ backend

## สิทธิ์การใช้งาน
โปรเจกต์นี้อยู่ในคลังโค้ดของผู้ใช้และมีจุดประสงค์เพื่อการศึกษาและตัวอย่างการใช้งาน หากต้องการใช้ซ้ำหรือเผยแพร่ต่อ กรุณาระบุแหล่งที่มาและชี้แจงสิทธิ์ตามที่เจ้าของโค้ดกำหนด
