# script_by_AI 🤖

คลังจัดเก็บและรวบรวมโปรเจกต์โค้ด HTML ที่ถูกสร้างและออกแบบโดยปัญญาประดิษฐ์ (AI) 

โปรเจกต์นี้สร้างขึ้นเพื่อเป็นพื้นที่รวบรวมผลงาน สคริปต์ หน้าเว็บเพจ และมินิแอปพลิเคชันที่เขียนด้วย HTML, CSS และ JavaScript โดยมี AI เป็นผู้ช่วยหลักในการเขียนโค้ดและออกแบบโครงสร้าง เพื่อเป็นแหล่งเรียนรู้ ไอเดีย และสามารถนำไปพัฒนาต่อยอดได้

---

## 🚀 สิ่งที่มีใน Repository นี้

* **Static Web Pages:** หน้าเว็บเพจแบบคงที่ (Landing Pages, Portfolios ฯลฯ)
* **Mini Web Apps:** แอปพลิเคชันขนาดเล็กที่ทำงานบนเบราว์เซอร์
* **UI Components:** ชิ้นส่วนอินเทอร์เฟซและลูกเล่นต่างๆ บนหน้าเว็บ
* **Experimental Scripts:** สคริปต์ทดลองการทำงานฟังก์ชันต่างๆ ที่สร้างโดย AI

## 💻 วิธีการนำไปใช้งาน

ไฟล์ส่วนใหญ่ในคลังนี้เป็นไฟล์ที่ทำงานบนฝั่งไคลเอนต์ (Client-side) คุณสามารถรันได้ง่ายๆ ดังนี้:

1. ดาวน์โหลดไฟล์จาก Repository นี้ (กด `Code` > `Download ZIP` หรือใช้คำสั่ง `git clone`)
2. แตกไฟล์ ZIP (หากดาวน์โหลดมา)
3. เข้าไปในโฟลเดอร์โปรเจกต์ที่คุณสนใจ
4. ดับเบิ้ลคลิกที่ไฟล์นามสกุล `.html` (เช่น `index.html`) ไฟล์จะถูกเปิดขึ้นมาทำงานบนเว็บเบราว์เซอร์ของคุณทันที โดยไม่ต้องติดตั้งเซิร์ฟเวอร์

## 🛠️ เทคโนโลยีและเครื่องมือ

* **Core Languages:** HTML5, CSS3, JavaScript (Vanilla)
* **AI Assistants:** โค้ดในคลังนี้ถูกสร้างหรือได้รับความช่วยเหลือจาก AI เช่น Gemini, ChatGPT, Claude เป็นต้น

## 📌 หมายเหตุ
